# Amaan Aslam Shaikh - Portfolio

A modern, responsive developer portfolio built with React, Tailwind CSS, and shadcn/ui components, featuring a beautiful Bento grid layout.

- **Modern Bento Grid Layout**: Clean, organized sections that adapt to different screen sizes
- **Dark Theme**: Sleek black and blue color scheme for a professional look
- **Responsive Design**: Optimized for desktop, tablet, and mobile devices
- **Smooth Animations**: Framer Motion animations for enhanced user experience
- **Interactive Components**: Hover effects and smooth transitions
- **Contact Form**: Functional contact form for potential clients
- **Professional Sections**: Hero, About, Skills, Projects, Experience, Education, and Contact

## üì¶ Installation

1. Clone the repository:
```bash
git clone <repository-url>
cd portfolio
```

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm run dev
```

4. Open your browser and navigate to `http://localhost:5173`

## üé® Customization

### Colors
The color scheme can be customized in `src/index.css` by modifying the CSS custom properties.

### Content
Update the content in each component file:
- `src/components/Hero.jsx` - Personal information and introduction
- `src/components/About.jsx` - About section and profile picture
- `src/components/Skills.jsx` - Technical skills and technologies
- `src/components/Projects.jsx` - Featured projects
- `src/components/Experience.jsx` - Work experience
- `src/components/Education.jsx` - Educational background
- `src/components/Contact.jsx` - Contact information and form

### Profile Picture
Replace `src/assets/profile.jpg` with your own profile picture.
